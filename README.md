# BC and B's - Professional Lawn Care Website

A modern, responsive website for BC and B's lawn care business in Conneaut, Ohio. Built with Next.js, React, and Tailwind CSS.

## 🌿 Features

✅ **Modern Design**
- Glassmorphism navbar with scroll detection
- Smooth animations with Framer Motion
- Organic & earthy theme with green color palette
- Fully responsive mobile design

✅ **Key Sections**
- **Sticky Navbar** - Glass effect navigation with mobile menu

## 🚀 Getting Started

### Prerequisites
- Node.js 16+ 
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone https://github.com/BroNel20/Bc-and-b-lawn-mowing.git
cd Bc-and-b-lawn-mowing
```

2. Install dependencies:
```bash
npm install
```

3. Run the development server:
```bash
npm run dev
```

4. Open [http://localhost:3000](http://localhost:3000) in your browser

### Build for Production

```bash
npm run build
npm start
```
